PROJECTS

DAY TRIPPER

Created at BASEHacks 2016, Day Tripper is a chatbot that simplifies planning for a short day trip, providing transit time and weather. Our team was awarded Best Implementation of the Gupshup Platform and was recognized as a Top Team by Wolfram Alpha.

GITHUB DEVPOST

REDIS PONG

Created at EVHacks II, Redis Pong is a variation of the traditional video game Pong. Players on two iOS devices can control paddles in a game of Pong coded in Java. A Redis server was used for communication between the controller iOS app and the Java game.

GITHUB DEVPOST

TWENTY

Submitted to the 2016 Congressional App Challenge, Twenty is an iOS app that assists heavy computer screen users in managing screen usage time. The app features alarms to take timed breaks, and also logs usage time.
