Projects

Day Tripper

Day Tripper is a chatbot developed using the Gupshup platform. Its goal is to simplify the planning process for the user when going out for a short day trip. The user simply enters their destination address, and the bot provides transit time and weather information in a simple, conversational interface. I built this with a friend at BASEHacks 2016, where we won Best Implementation of the Gupshup Platform and were recognized as a Top Team by Wolfram Alpha.
