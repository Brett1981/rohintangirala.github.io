Portfolio

I am always working on new projects that demonstrate my passion for coding.
Here is a selection of a few that I have been working on:

LastStandVR

Created at HSHacks III, LastStandVR is a first person shooter that passes a phone's rotational vector data through a Redis server to control gameplay in Unity. It features both an Android and an iOS version of the controller app. I primarily worked on the iOS controller app. The game itself involves shooting monsters that spawn all around the player. This project was our team's effort to create a more accessible VR-like experience.

  • Source on Github
  • Read on Devpost

Optimized Place Search

Created at CodeDay Bay Area, Optimized Place Search is a web app that takes a user's current location and desired destination, such as a store or restaurant, then informs the user whether the user will arrive during business hours. I was awarded Best Application at CodeDay Bay Area for the project.

  • Source on Github

Twenty

Submitted to the 2016 Congressional App Challenge, Twenty is an iOS app that assists heavy computer screen users in managing screen usage time. The app features alarms to take timed breaks, and also logs usage time.

  • Source on Github
  • Watch the Video

Redis Pong

Created at EVHacks II, Redis Pong is a variation of the traditional video game Pong. Players on two iOS devices can control paddles in a game of Pong coded in Java. A Redis server was used for communication between the controller iOS app and the Java game.

  • Source on Github
  • Read on Devpost

Day Tripper

Created at BASEHacks 2016, Day Tripper is a chatbot that simplifies planning for a short day trip, providing transit time and weather. My team was awarded Best Implementation of the Gupshup Platform and was recognized as a Top Team by Wolfram Alpha.

  • Source on Github
  • Read on Devpost
