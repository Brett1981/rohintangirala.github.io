Day Tripper

Created at BASEHacks 2016, Day Tripper is a chatbot that simplifies planning for a short day trip, providing transit time and weather. My team was awarded Best Implementation of the Gupshup Platform and was recognized as a Top Team by Wolfram Alpha.

  Github   Devpost

Redis Pong

Created at EVHacks II (part of MLH Local Hack Day III), Redis Pong is a variation of the traditional video game Pong. Players on two iOS devices can control paddles in a game of Pong coded in Java. A Redis server was used for communication between the controller iOS app and the Java game.

  Github   Devpost

Twenty

Submitted to the 2016 Congressional App Challenge, Twenty is an iOS app that assists heavy computer screen users in managing screen usage time. The app features alarms to take timed breaks, and also logs usage time.

  Github   Video

Optimized Place Search

Created at CodeDay Bay Area, Optimized Place Search is a web app that takes a user's current location and desired destination, such as a store or restaurant, then informs the user whether the user will arrive during business hours. I was awarded Best Application at CodeDay Bay Area for the project.
